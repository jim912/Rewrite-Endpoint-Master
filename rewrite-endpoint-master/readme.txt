== Description ==

You can add custom rewrite endpoints from admin page.


== Installation ==

1. Upload `rewrite-endpoint-master` to the `/wp-content/plugins/` directory
1. Activate `Rewrite Endpoint Master` through the 'Plugins' menu in WordPress
1. Add Rewrite Endpoint through the 'Endpoints' menu child of'Settngis' menu
